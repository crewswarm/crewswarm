Common questions

How do I report a bug?

Open an issue on GitHub. Include your OS, Node version, the command you ran, and the full error output. The faster you give us a repro, the faster we fix it.

Open a GitHub issue →

Is crewswarm free?

Yes. crewswarm is open-source and free to run. You supply your own API keys — we don't proxy your requests or charge for model usage. There's no hosted tier that gates features.

Get started →

What LLM providers are supported?

Anthropic Claude, OpenAI, Groq, Gemini, and any OpenAI-compatible endpoint. Each agent in a crew can use a different model. Mix and match based on cost, speed, or capability.

See all supported models →

How do I contribute?

Read CONTRIBUTING.md first — it covers branch conventions, commit style, and how to run the test suite locally. PRs that include tests get reviewed faster.

Read CONTRIBUTING.md →

Where can I get help?

Start with the docs — most setup questions are answered there. If you're stuck on something specific, open a GitHub Discussion so the answer is findable by the next person too. For urgent issues, email works.

Read the docs  ·  GitHub Discussions  ·  info@crewswarm.ai

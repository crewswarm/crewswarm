Installation

crewswarm is a local-first system. Ensure you have Node.js 20+ installed.

1. Clone the repository

git clone https://github.com/crewswarm/crewswarm.git
cd crewswarm

2. Install dependencies & Setup

npm install
bash install.sh

3. Start all services

npm run restart-all

Once running, the dashboard is accessible at http://127.0.0.1:4319.

Execution Modes

Direct API

Agents call LLMs (Groq, Anthropic, OpenAI) directly. Best for analysis and planning tasks that don't require file system access.

OpenCode (Default)

Recommended for coding. Agents run in a sandboxed session with full system access. OpenCode routes reasoning to models like GPT-5.3 Codex, Claude 3.5, and Gemini.

Claude Code CLI

Integrates directly with Anthropic's native `claude -p` CLI for tasks that benefit from its specific codebase reasoning capabilities.

Cursor CLI

Integrates directly with the Cursor editor's CLI, leveraging its internal codebase indexing and MCP models.

Best Practices

  • Update the Brain

    Maintain memory/brain.md in your project. This file is the primary context source for all agents. Use it to store project rules and architectural decisions.

  • Leverage the PM Loop

    Don't manage tasks manually. Create a ROADMAP.md and let crew-pm handle the decomposition. This enables autonomous, multi-step progress.

  • Task Granularity

    The smaller the task, the higher the success rate. If a task is complex, ask crew-pm to break it down further before dispatching it to crew-coder.

Technical Manual

Engineering-First Terminals

The ATAT Protocol

The CLI uses the proprietary ATAT Protocol. It's 10x more token-efficient than JSON-RPC, reducing context window bloat and eliminating the escaping errors that crash sequential AI agents.

Read the Spec →

3-Tier LLM Routing

Bypass the "one model fits all" trap. crew-cli uses Gemini 2.5 Flash for cheap routing, Claude Sonnet 4.6 for architecture planning, and a pool of Grok 3/DeepSeek V3 workers for parallel execution.

Parallel Worker Pool

Why wait for one agent to finish? crew-cli executes independent tasks concurrently. In benchmarks, we achieve a 2.96x wall-clock speedup over sequential implementation cycles.

LSP Self-Healing

The CLI integrates the Language Server Protocol directly into the sandbox. Agents receive real-time compiler errors and fix their own syntax/type issues before you ever see the code.

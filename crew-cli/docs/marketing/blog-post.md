# Announcing crew-cli: The Multi-Agent Orchestrator for Your Terminal

We are excited to announce `crew-cli` v0.1.0, a command-line tool that brings the power of CrewSwarm directly to your terminal. 

Whether you need a dedicated frontend specialist, a bug-fixing expert, or a product manager to help plan out a massive refactor, `crew-cli` lets you seamlessly interact with specialized AI agents while keeping complete control over your codebase.

## The Problem

Most AI coding tools force you into a specific paradigm. You either use a generic assistant that tries to do everything, or you are locked into a proprietary IDE. Furthermore, applying AI-generated code directly to your file system is risky and often requires tedious copy-pasting or manual `git checkout`s to undo hallucinations.

## The crew-cli Solution

`crew-cli` solves this by combining specialized multi-agent orchestration with a **Cumulative Diff Sandbox**.

### 1. Multi-Agent Orchestration
By typing `crew chat "Fix the auth bug"`, the Orchestrator automatically analyzes your intent and routes your task to the best specialist (e.g., `crew-fixer`). You get expert-level help without having to prompt engineer a generalist model.

### 2. The Sandbox
All file modifications generated by the agents are staged in the Sandbox (`.crew/sandbox.json`), not applied directly to your disk. You can preview changes using `crew preview`, compare different approaches using `crew branch`, and when you are satisfied, run `crew apply` to commit the changes to your working directory.

### 3. Bring Your Own Tokens
We know AI can be expensive. `crew-cli` includes a `crew auth` command that automatically discovers existing session tokens from tools you already use (like Claude Code, Cursor, or your Google ADC credentials), allowing you to leverage local models and existing subscriptions seamlessly.

## Get Started

Install `crew-cli` today:

```bash
npm install -g @crewswarm/crew-cli
```

Check out our [GitHub repository](https://github.com/crewswarm/crew-cli) for documentation and examples. Let's build the future of software development, together!

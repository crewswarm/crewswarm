**Name:** crew-cli
**Tagline:** The multi-agent orchestrator for your terminal

**Description:**
`crew-cli` brings specialized AI agents (coders, QA, designers) directly to your terminal. It features a unique Cumulative Diff Sandbox that stages all AI-generated code changes, letting you preview diffs, branch alternatives, and `apply` safely when ready. New: `crew review --strict` for CI gates, headless JSONL artifact output (`--headless --json --out`), context budget controls (`--max-context-tokens`), and `crew mcp doctor` preflight checks.

**Topics:** Developer Tools, Artificial Intelligence, Command Line

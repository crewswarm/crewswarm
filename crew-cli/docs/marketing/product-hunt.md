**Name:** crew-cli
**Tagline:** The multi-agent orchestrator for your terminal

**Description:**
`crew-cli` brings specialized AI agents (coders, QA, designers) directly to your terminal. It features a unique Cumulative Diff Sandbox that stages all AI-generated code changes, allowing you to preview diffs, branch alternative implementations, and `apply` safely only when you're ready. Bring your own API tokens or let `crew-cli` auto-discover your existing local sessions to save costs.

**Topics:** Developer Tools, Artificial Intelligence, Command Line

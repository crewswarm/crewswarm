Common Questions

How is this different from Aider or Cursor?

Cursor is an IDE; Aider is a single-agent CLI. `crew-cli` is an orchestrator that routes your tasks to a crew of specialists. It also uses a unique cumulative sandbox, so you can branch implementation ideas without touching your git history until you're ready.

Do I need to pay for a subscription?

`crew-cli` itself is open-source. It can use your existing API keys or auto-discover local tokens from Claude Code and Cursor to save you money. You only pay the model providers for what you use.

What models are supported?

Through the CrewSwarm gateway, we support Claude 3.5 Sonnet, GPT-4o, DeepSeek, and Grok. The orchestrator chooses the best model for the specific sub-task (e.g., PM for planning, Coder for implementation).

Is my data private?

Absolutely. `crew-cli` is local-first. Your code and session data stay on your machine in the `.crew/` directory. We only send the minimum context required to your chosen LLM provider, and you have full control over what gets shared with your team via the `crew privacy` command.

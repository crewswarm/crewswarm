🕷️ Swarm Dashboard

Monitor and manage your OpenCode swarm agents in real-time.

Quick Start

  • View Agents: Check "Agent Overview" for live stats
  • Create Session: Use "Quick Actions" to spawn a new agent
  • Monitor Activity: Watch "Live Activity Feed" for real-time updates
  • Manage Memory: Use "Shared Memory" to read/write agent data

Dashboard Features

  • Agent Overview: Total agents, roles, types, tasks, tokens, cost
  • Agent Breakdown: Count by type (Build, Explore, General, Other)
  • Activity Timeline: Hourly agent activity chart
  • Token Distribution: Token usage by agent type
  • Live Activity Feed: Real-time tool calls and task updates
  • Active Sessions: Grid of running agents with controls
  • Shared Memory: View and edit cross-agent shared state

Session Controls

  • Copy ID: Copy session ID to clipboard
  • Details: View full session information
  • Kill: Terminate the agent (use with caution!)

Demo Mode

Click the 🎭 button to enable demo mode with simulated agent activity (useful for testing UI).

Keyboard Shortcuts

  • R: Refresh dashboard
  • Esc: Close modal dialogs
  • Ctrl+K: Focus search/filter

Core Features

📡

OpenCrew RT

WebSocket Realtime Protocol for agent coordination. Channels: command, assign, status, issues, handoff, done, reassign. At-least-once delivery with durable JSONL logs. Token auth, rate limiting, and TLS support.

🧠

Shared Memory

Persistent cross-session state stored in ~/.ai-swarm-memory/. Agents share context, lessons, and project knowledge. Supports namespaces, JSONL logs, and atomic writes.

🐝

Multi-Agent Orchestration

Coordinated agent swarms with role-based dispatch: Orchestrator, PM, QA, Fixer. Automatic task assignment, issue escalation, and handoff flows.

⚡

Real-Time Dashboard

Live monitoring interface with Agent Overview (totals, tokens, costs), Activity Timeline charts, Token Distribution by type, Live Activity Feed, Session cards with controls, and Shared Memory panel. Demo mode for testing.

🔧

Code Execution

Production-grade code execution with strict validation, file verification, and automatic retry logic.

🔒

Strict Verification

Every code change is validated. No more "I'll help you" responses that don't produce actual files.

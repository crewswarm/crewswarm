Keyboard Shortcuts Guide

Keep the core editing and navigation actions one keystroke away. Reopen this panel anytime from the title bar.

Tip Prefer familiar conventions, avoid overriding browser defaults unless the payoff is clear, and always keep a clickable UI path alongside the shortcut.
Cmd/CtrlKInline AI

Open the Monaco inline assistant from the current cursor position.

EnterSend chat

Submit the main chat input without leaving the keyboard.

ShiftEnterNew line

Insert a line break in chat instead of sending immediately.

EscDismiss overlays

Close inline chat, diff review, or any temporary modal state.

Cmd/CtrlZUndo

Use Monaco's built-in undo stack while editing the current file.

Cmd/CtrlShiftZRedo

Replay an undone change in the active Monaco editor.

Cmd/Ctrl/Toggle comment

Comment the current line or selected block while editing.

Cmd/CtrlSSave file

Commit the current edit to disk before switching context.

Best Practices

  • Reserve the easiest shortcuts for high-frequency actions such as opening files, saving, and invoking AI help.
  • Match platform habits when possible so users can transfer muscle memory from editors like VS Code.
  • Keep destructive commands out of single-key bindings unless there is a clear confirmation step.
  • Expose every shortcut in tooltips, menus, or guides so discoverability does not depend on memory alone.
  • Design shortcuts to work with one hand where practical, especially for actions users repeat dozens of times per session.
